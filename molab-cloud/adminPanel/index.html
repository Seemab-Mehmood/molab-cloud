Deployment note

Hospital and patient records live in this server's SQLite database. This panel is gated by (1) a private, unlisted URL path, (2) admin email/password auth, and optionally (3) a shared secret header. For real production use, also put this behind a VPN or IP allowlist, enable 2FA, and rotate the URL path and secret key periodically.
